## Usage  
> Start app 
```
npm run start
```
> Start app by electromon  
```
npm run s:dev
```
> Build for all platform
```
npm run build:all
```
> Build for Windows
```
npm run build:win
```
> Build for Linux
```
npm run build:linux
```
> Build for Mac
```
npm run build:mac
```
